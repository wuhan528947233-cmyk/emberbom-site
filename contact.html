Safe contact

Contact EmberBOM

Email support@emberbom.com for product, purchase, or refund questions.

Safe information to send

  • EmberBOM version
  • Windows x64 or Linux x64
  • Workflow stage
  • A structured EmberBOM error code, if one was displayed
  • A description written in your own words
  • A Paddle transaction number for purchase or refund questions

Never send project material

Do not attach or paste source code, project files, paths, filenames, hashes, repository addresses, compile commands, linker maps, build logs, terminal screenshots, archives, SBOMs, HTML reports, scan JSON, or review YAML.

Support boundary

The license includes product documentation and generally released corrections. It does not include project inspection, build configuration, custom rules, consulting, or a guaranteed response time.
